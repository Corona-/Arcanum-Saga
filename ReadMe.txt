RPGの問題点

レベル上げは簡単であってはならない
バランスはシビアに、レベル上げに戦略性・テクニックが必要だといい
自由度が足りない
　⇒経験値システムでよいところは出来なかったことが出来るようになること
　⇒自由があるからレベルをあげて行動範囲が広がることが良い
プレイヤーが自分のレベルに合わせて自由に行動すればバランスの問題は発生しない

戦闘以外のシステムがない

ダンジョンは敵の本拠地であるわけで準備もなしに突入するのは危険ではないか？
⇒盗賊あたりに下調べさせるか、人から情報を集めたい
⇒ダンジョンのマップが手に入るならなお良い　（だが敵の本拠地のマップがあるのだろうか？）
⇒ダンジョンの中でも出来るだけ慎重に行動すべきではないか？
　⇒敵に囲まれる危険もあるだろう

ゲームを進めるために必ずしも戦闘をする必要がないのがいい？
冒険者なのに冒険していない

トラップや仕掛けを回避しても経験値はなし、宝物を首尾よく手に入れても経験値は入らない
イベントをクリアしても経験値は入らない
⇒冒険者の目標は宝を持ち帰るのが目的だから宝物を持ち帰ったら経験値が入るのは当然

戦闘システムの問題

プレイヤーにリソースを与えすぎてはいけないし、与えなさ過ぎてもいけない
⇒例：トルネコの大冒険の経験値とアイテムの量
⇒限られたアイテムをどう使うか？いかに効率よく経験値を稼ぐか？

使えきれないほどMPがあるのはおかしい
⇒回復魔法使い放題なら適当に戦って回復しつつ進めばいいだけ
アイテム使い放題というのもおかしい
⇒もてる数が多すぎる
⇒回復魔法と同等の効果のものもあったりする（回復魔法すらいらない？）

ステータス異常がぬるい
⇒Wizardryの状態異常は非情に危険
最近の状態異常はアイテムひとつで回復できてしまう

ひとつの戦闘をいかに乗り切るか、になってしまっている

戦闘の戦略性の低さ
　
MPを使って攻撃するのならば普通に攻撃して回復魔法を使うより多いダメージを与えれなければ意味がない？
例外：攻撃魔法で無いと倒せない敵がいる、直接攻撃でも倒せるが見方の被害が大きくなるため魔法で回避するケース

考える要素が少なくなっているのが問題である

⇒潜入しているのに敵を倒して経験値を得なければならない無jyん
⇒薬草を一刻も早く取ってきてほしいのに戦闘をしたほうが良い
　⇒薬草を早く持ってきても経験値が得られない

魔法のエフェクトが長いのは良くない
戦闘で演出が多すぎる
分単位の戦闘になるのは良くない？秒程度にしろ？
⇒それよりも演出が長すぎるのは良くない

セーブシステム
なぜ日本のゲームは常時セーブにしないのか？
⇒ゲームをやめたいときにやめることが出来ない
⇒まとまった時間の取れないプレイヤーには致命的な問題
⇒連続プレイ時間が１時間を越えるようなら常時セーブを採用すべき
⇒いつでもセーブできると重要なイベントの手前でセーブして安全なプレイが出来てしまう？
　⇒別に問題ないはず、少しずつ先のことを理解し、攻略していくので
　⇒それよりもセーブポイントから離れすぎてやり直しを余儀なくされるほうが問題

ミニゲームの問題
ミニゲームがゲーム本編の面白さに結びついていない
ゲーム本編をやっているプレイヤーにとってミニゲームは邪魔？
戦闘とイベントの繰り返しを避けるという安易な目的のために挿入された場合が多い

ダンジョンの攻略
ダンジョンの攻略はミニパズルを解くということに等しい
⇒何故ダンジョンに仕掛けられたパズルを解かなければいけないのか？

ダンジョンの罠はより精巧に綿密に作られていなければならない
⇒そして引っかかった相手は死ぬ

プレイヤーに求められるのは下調べと情報集め
⇒偵察対を派遣して下調べという手も有る
⇒敵の本拠地に何も知らずに侵入するのは無謀すぎる

ダンジョンに対抗できる人材を用意すること：盗賊
開錠、罠探知、罠解除、聞き耳、隠れ身、奇襲攻撃、探索（隠し扉や隠れた敵）、盗み、スリ
盗賊は戦士ほど強くなく、魔法を使うことが出来ないが、ダンジョン攻略には必須である
⇒戦闘中にアイテムを盗むというのもおかしい、死体になってから剥ぎ取ればよい

耐久戦の要素
⇒限られた戦力を駆使して、どこまで進めるか
⇒いかにして目的地までたどり着けるか
⇒一定の場所を拠点として、そこから行動の範囲を広げていく
⇒うまくなればなるほど行動の範囲が広がっていく
⇒HPやMPが尽きたから本拠地に何度も帰るというプレイスタイルが珍しくなった
⇒遠くに行くにつれて、より強いモンスターが出るように設定されている
　⇒ダンジョンの奥に行くにつれて…と同じである
⇒簡単に次の目的地にたどり着けないようになっている
⇒何故なくなったか⇒ストーリーが重視されるようになったから
　⇒進めなければストーリーが進まないから
　⇒プレイヤーが快適なスタイルを望んでいたから
⇒出来るだけ体力や魔力を温存しつつ進む必要がある
　⇒同じ敵を倒すにしても出来るだけうまく倒す必要がある
⇒耐久戦の要素があると目的地までの戦闘に緊張感がある
